# AI Study Assistant with RAG (Azure OpenAI)

## Overview
A full-stack AI-powered study assistant that allows users to upload notes and ask questions. 
Implements Retrieval-Augmented Generation (RAG) to provide context-aware answers.

## Features
- Chat-based AI assistant
- Document upload (PDF/text)
- Semantic search over notes
- Context-aware responses using Azure OpenAI

## Tech Stack
- Frontend: Next.js
- Backend: .NET Core Web API
- Database: SQL Server
- AI: Azure OpenAI (GPT + embeddings)

## How It Works
1. Upload documents
2. Text is chunked and embedded
3. Relevant chunks are retrieved
4. Sent to AI for contextual answers
